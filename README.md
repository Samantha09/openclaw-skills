# OpenClaw Skills Collection

KimiCode 的 OpenClaw 技能集合。

## 技能列表

| 技能 | 描述 | 路径 |
|------|------|------|
| java-dev-assistant | Java 开发助手 - 功能开发、Bug 修复、提交规范、测试生成、文档记录 | [java-dev-assistant/](java-dev-assistant/) |
| java-code-reviewer | Java 代码审查助手 - 代码评审、问题发现、质量评估、安全审查 | [java-code-reviewer/](java-code-reviewer/) |

## 使用方法

将技能目录复制到 OpenClaw 的 skills 目录：

```bash
cp -r java-dev-assistant ~/.openclaw/workspace/skills/
```

## 添加新技能

1. 在仓库根目录创建新的技能文件夹
2. 按照技能规范创建 SKILL.md 和必要资源
3. 更新本 README 的技能列表
